1 · Install trelly

npm install -g trelly   # or: brew install brandonkramer/tap/trelly

2 · Log in once

trelly auth setup
trelly auth login

3 · Wire your agent (Cursor / Claude)

Add to ~/.cursor/mcp.json or your Claude MCP config:

"trelly": {
  "command": "trelly-mcp",
  "env": { "TRELLO_PROFILE": "default" }
}

Your agent can now list boards, move cards, comment, and attach PRs on this board. Docs: github.com/brandonkramer/trelly · npm
